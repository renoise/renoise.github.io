Module xLib.classes.xEffectColumn

Use xEffectColumn to create 'virtual' renoise.EffectColumn objects .

Unlike the renoise.EffectColumn, this one can be freely defined, without the need to have the line present in an actual song

You create an instance by feeding it a descriptive table in the constructor. All string-based values are automatically converted into their numeric counterparts.
